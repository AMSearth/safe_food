# Safe Food

**Eat Smart. Live Healthier.**

Safe Food is a lightweight web application that allows users to create an account, log their medical information (age, weight, height, conditions, allergies), and receive food recommendations and personalized food analyses.

## Features
- **Frontend**: Clean, premium green aesthetic using robust Glassmorphism UI patterns. No frameworks, strictly Vanilla HTML/CSS/JS.
- **Backend**: Built atop a fast, responsive Python Flask core.
- **Database**: SQLite3 database seamlessly manages patient data and food history.
- **Analysis**: Submit a food item (text or image) to understand its nutritional weight considering your personal health data.

## Installation

1. Clone the repository.
   ```bash
   git clone <your-repo-link>
   cd safe_food
   ```
2. Setup Virtual Environment (Recommended)
   ```bash
   python -m venv venv
   # Mac / Linux
   source venv/bin/activate
   # Windows
   venv\Scripts\activate
   ```
3. Install Dependencies
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Server
   ```bash
   python app.py
   ```
5. View Application:
   Navigate to `http://127.0.0.1:5000` via your web browser.
